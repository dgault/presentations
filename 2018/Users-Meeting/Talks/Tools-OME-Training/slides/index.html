Growing ecosystem

Aims

  • Provide reusable training materials
  • Provide resources for setting up training sessions
  • Have resources under Version Control
  • Have resources publicly available

Materials

  • Based on “workflows”
    • Link components of the OME ecosystem
    • Closer to “real” users day-to-day tasks
  • Available for different audiences
    • Biologists
    • Facility managers
    • Image Analysts
    • ...
