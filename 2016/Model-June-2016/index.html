Model Changes

New: Folders

  • Folders may contain:
    Folders, ROIs, Images
  • Folder hierarchy is a strict tree:
    a Folder may have only one parent

Model Changes

Folders and Images

  • Images are associated with Folders,
    • directly via ImageRef or
    • indirectly via ROIRef
  • OMERO.insight, .web initially support only the latter
