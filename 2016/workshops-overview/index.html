Workshops Overview

OME Users Meeting, Wednesday 1st June 2016

Session 1, Room2: Infrastructure for OMERO

An overview of the tools and workflows used by the OME for developing OMERO, and for running OMERO servers in production at scale.

Session 2, Room 2: Extensible Import

Different ways of importing data into OMERO, with a focus on large datasets with complex metadata.

Session 3, Room 2: Interacting with your data

How to manage and interrogate large datasets in OMERO using the command line tools, scripts and other interactive applications.
